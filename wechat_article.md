## Brainless：让 Claude Code 拥有永久记忆

我写了一个 Claude Code Skill，叫 **Brainless**（没脑子）。

名字很反直觉，但逻辑很直接：**你不需要脑子，因为它帮你全记着。**

装上之后，Claude Code 的行为会发生两个根本性的变化：

### 变化一：遇到报错，它会先「回忆」

以前：报错 → 开始调试 → 30 分钟后修好 → 下次再来一遍

现在：报错 → **自动搜索知识库** → 发现三周前修过 → 3 秒内给出方案

### 变化二：解决问题后，它会自动「记笔记」

不需要你手动输入任何命令。Claude 自动检测到问题被解决了，自动分类（是编译错误？运行时崩溃？配置问题？），自动用专属模板写入知识库。

**完全无感。你只管写代码，它在后台默默记住一切。**

---

## 不止是记错误 —— 它什么都记

Brainless 有 13 个分类、6 套专属模板，覆盖开发者日常遇到的所有场景：

**开发日常：**
- 编译报错 → 记录错误信息 + 根因 + 环境 + 修复步骤
- 运行时崩溃 → 同上
- 依赖冲突 → 同上
- 配置问题 → 同上

**安全研究 / CTF：**
- CTF 题解 → 记录**错误尝试**（这很关键）+ 关键突破 + flag + 经验总结
- 逆向笔记 → 记录目标信息、保护措施、分析过程、可识别模式
- 漏洞利用 → 记录前置条件、绕过方法、代码模板、常见坑点

**技巧和工具：**
- 实用技巧 → 使用场景、操作方法、原理
- 工具用法 → 具体命令、预期输出、隐藏技巧

注意 CTF 模板有一个特别的字段叫「Wrong Approaches」—— **记录你尝试了什么但失败了**。这可能是整个系统最有价值的部分。因为下次遇到类似题目时，你不只知道正确答案，还知道哪些路是死路。

## 三级搜索：知识库再大也不烧 Token

Claude Code 的上下文窗口是有限的。如果知识库有 1000 条记录，每次搜索都把所有内容读一遍，光搜索就要烧掉 2 万个 token。

Brainless 用了一个三级搜索策略：

```
第一级：读 _cache.json（轻量 JSON 缓存）  → ~50 tokens
第二级：读分类子索引                       → ~20 tokens
第三级：读完整条目                         → 仅匹配时才读
```

**对比数据：**

| 知识库大小 | 暴力全读 | Brainless | 节省 |
|-----------|---------|-----------|------|
| 100 条 | ~2,000 tokens | ~800 tokens | 60% |
| 500 条 | ~10,000 tokens | ~3,000 tokens | 70% |
| 1000 条 | ~20,000 tokens | ~5,000 tokens | 75% |

知识库越大，优势越明显。**你的知识在增长，但搜索成本几乎不变。**

## 它还会分析你的弱点

`/brain-stats` 不只是给你看「你记了多少条」。它会分析模式：

- **CTF 弱点**：哪类题你失败最多？pwn 题总是栽在堆利用上？web 题总是忽略 SSRF？
- **反复踩的坑**：哪类错误的 hit_count 最高？（高 hit_count 意味着你一直在犯同样的错）
- **逆向难点**：ARM 架构总比 x86 多花三倍时间？Ghidra 对某类壳的反编译总不对？
- **你的强项**：哪些问题一次解决、再没出现？这是你的舒适区。

**它不只是一个数据库。它是你的技能雷达图。**

## 间隔复习 —— 对抗遗忘曲线

`/brain-review` 用测验的形式帮你复习旧知识：

1. 给你看一个过去记录的**问题描述**
2. 问你：「还记得怎么解决的吗？」
3. 翻牌揭晓答案

优先复习：
- 很久没被回忆的旧条目（可能已遗忘）
- 标记为「未解决」的 CTF 题（换个思路再试）
- 高价值技巧（定期温习防止生疏）

```bash

cd brainless-skill
bash install.sh
```

## 一个哲学问题

我们总觉得「记住东西」是一种能力。但在 AI 辅助编程的时代，记忆力反而是最不值钱的技能 —— 因为存储和检索可以被完美自动化。

真正值钱的是：
- **判断力** —— 知道该记什么，不该记什么
- **模式识别** —— 从一堆记录中看出规律
- **创造力** —— 用已知的知识组合出新的方案

Brainless 帮你把记忆外包了。你省下来的脑子，拿去想更重要的事。

**Be brainless. Be free.**
